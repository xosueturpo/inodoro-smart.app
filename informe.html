1. Prototipo y visión de producción

2. Problema y justificación

En el Perú, la mayoría de baños domésticos siguen siendo sistemas mecánicos convencionales: sin sensores, sin control remoto y sin información del estado del tanque. Aunque en el mercado internacional ya existen inodoros inteligentes, su precio suele ser elevado y su importación limita el acceso de gran parte de la población nacional.

Inodoros Fuertes es una propuesta académica y de producto peruano para acercar la automatización del baño a hogares reales: descarga y recarga asistidas, control por aplicación (Bluetooth / WiFi), comandos de voz y detección por proximidad.

El propósito no es solo demostrar que la tecnología existe, sino mostrar que puede desarrollarse desde una marca peruana, a precios más accesibles, para modernizar hogares en la era de la tecnología sin depender únicamente de equipos importados de lujo.

3. Objetivos

Objetivo general

Diseñar, construir y documentar un prototipo funcional de inodoro inteligente controlado por microcontroladores y una aplicación móvil multiplataforma.

Objetivos específicos

  • Automatizar descarga y recarga del tanque con sensores de nivel y proximidad.
  • Permitir control remoto desde una app Flutter (BLE y red local).
  • Integrar comandos de voz y una interfaz clara para el usuario final.
  • Validar el flujo electrónico y de software sobre un prototipo funcional.
  • Plantear una visión de producto nacional competitivo frente a soluciones importadas.

4. Arquitectura del sistema

El sistema se organiza en tres capas. La aplicación no se comunica directamente con el Arduino: todo pasa por el ESP32.

Capa Componente Función
Aplicación Flutter / Dart Interfaz, BLE, LAN, voz, modo demo
Comunicaciones ESP32 Puente BLE / WiFi / HTTP / LCD / UART
Control físico Arduino UNO Válvula, tapa, bomba, sensores, buzzer, LEDs

Organización del código

  • codigo_ArduinoUNO/ — firmware del Arduino UNO
  • codigo_ESP32/ — firmware del ESP32
  • Resto del repositorio (lib/, android/, ios/…) — aplicación Flutter

5. Componentes electrónicos

Materiales y componentes electrónicos utilizados en el prototipo:

Componente Función Conexión / notas
Arduino UNO Control de actuadores y sensores Firmware en codigo_ArduinoUNO/
ESP32 BLE, WiFi, HTTP, LCD, puente UART Firmware en codigo_ESP32/
Sensor de nivel de agua Detectar tanque lleno / vacío ADC en A0 (umbral filtrado > 400)
HC-SR04 (descarga) Proximidad para iniciar descarga Trig D6 / Echo D7
HC-SR04 (tapa) Proximidad para abrir tapa Trig D2 / Echo D3
Servo válvula (≈11 kg) Abrir / cerrar descarga D5, hold PWM continuo
Servo tapa Abrir (85°) / cerrar (0°) A3 (pin digital 17)
Relé + bomba / electroválvula Recarga del tanque D10
Buzzer activo Señal sonora de estado D4
74HC595 + LEDs RGB Estado visual (reposo / ciclo / error) D11 clock, D12 latch, D13 data
LCD I2C (vía ESP32) Mensajes de estado al usuario Gestionado por ESP32
Fuente y cableado UART Alimentación y enlace UNO ↔ ESP32 UNO D8/D9 · 9600 baud · GND común
Dispositivo Android / iOS Aplicación de control Flutter (BLE + LAN + voz)

6. Software y funcionamiento

Aplicación móvil (Flutter)

  • Conexión por Bluetooth LE (configuración WiFi del ESP32).
  • Conexión por red local (mDNS / HTTP).
  • Comandos: descarga, recarga, abrir/cerrar tapa.
  • Comandos de voz y modo demo sin hardware.

Firmware

  • Loop cooperativo en el Arduino UNO (sin delay largos).
  • Máquinas de estado para descarga y recarga, con watchdogs.
  • Filtro de ruido en el sensor de agua (mediana + confirmaciones).

Impacto esperado

Al plantear una marca peruana con enfoque en resistencia y precio accesible, el proyecto busca acercar la modernización del baño a más hogares: no como un lujo importado, sino como una mejora cotidiana.

7. Conclusiones

  • Se logró un prototipo funcional que integra sensores, actuadores, ESP32 y una aplicación multiplataforma bajo la marca Inodoros Fuertes.
  • El prototipo valida la electrónica y el flujo de uso; la previsualización de producción orienta el siguiente paso de diseño industrial.
  • Aunque ya existen smart toilets en el mundo, hay oportunidad para una alternativa nacional más asequible que modernice hogares peruanos.
  • Trabajo futuro: carcasa sanitaria, certificación eléctrica, costos de BOM y pruebas de durabilidad en uso real.
